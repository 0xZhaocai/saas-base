# 极简 Blog - 产品需求文档 (PRD)

> **模块目的**: 作为 SaaS 底座的示例模块，演示规范开发流程，验证架构设计

---

## 1. 产品概述

### 1.1 背景
为验证 SaaS 底座的扩展性和开发友好度，创建一个极简 Blog 模块作为示例。

### 1.2 目标用户
已登录的 SaaS 用户

### 1.3 核心价值
- 为开发者提供**完整的开发参考案例**
- 验证**新模块对底座的入侵程度**
- 验证**底座对新模块的友好程度**

---

## 2. 用户故事

### 2.1 文章管理

| ID | 用户故事 | 优先级 |
|----|---------|--------|
| US-01 | 作为用户，我可以**查看文章列表**，以便浏览所有文章 | P0 |
| US-02 | 作为用户，我可以**创建新文章**，以便发布内容 | P0 |
| US-03 | 作为用户，我可以**查看文章详情**，以便阅读完整内容 | P0 |
| US-04 | 作为用户，我可以**编辑自己的文章**，以便修改内容 | P0 |
| US-05 | 作为用户，我可以**删除自己的文章**，以便移除不需要的内容 | P0 |

## 4. 页面结构

```
/blog              → 文章列表页
/blog/new          → 新建文章页
/blog/:id          → 文章详情页
/blog/:id/edit     → 编辑文章页
```

---

## 5. 数据模型

### 5.1 文章表 (post)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT | 主键 |
| title | TEXT | 标题 (必填) |
| content | TEXT | 内容 (必填) |
| authorId | TEXT | 作者 ID (关联 user) |
| createdAt | TIMESTAMP | 创建时间 |
| updatedAt | TIMESTAMP | 更新时间 |

---

## 6. 多语言需求

需要翻译的文本:

| 键 | 说明 |
|----|------|
| nav.blog | 导航菜单 |
| blog.title | 页面标题 |
| blog.newPost | 新建文章按钮 |
| blog.editPost | 编辑文章 |
| blog.deletePost | 删除文章 |
| blog.deleteConfirm | 删除确认 |
| blog.titlePlaceholder | 标题输入框占位符 |
| blog.contentPlaceholder | 内容输入框占位符 |
| blog.save | 保存按钮 |
| blog.cancel | 取消按钮 |
| blog.noContent | 暂无文章 |
| blog.by | 作者前缀 (by xxx) |
